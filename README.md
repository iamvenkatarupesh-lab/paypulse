# PayPulse

A cloud-native, microservice-based payment application built to demonstrate end-to-end DevOps practices.

## Stack
- **Frontend:** Next.js (React)
- **Backend:** Node.js + Express
- **Database:** PostgreSQL
- **Containers:** Docker, Docker Compose
- **Orchestration:** Kubernetes (AWS EKS)
- **IaC:** Terraform
- **CI/CD:** GitHub Actions
- **Observability:** Dynatrace (OneAgent, APM, Logs, Dashboards)
- **Reverse Proxy:** NGINX
- **Cloud:** AWS (EKS, RDS, ECR, ALB, Route53, ACM, IAM, VPC)

## Repository Layout

- frontend/        Next.js application
- backend/         Node.js Express API
- infra/terraform  AWS infrastructure as code
- infra/kubernetes Helm charts and raw manifests
- .github/workflows GitHub Actions CI/CD pipelines
- docs/            Architecture decisions and diagrams
- scripts/         Developer helper scripts
